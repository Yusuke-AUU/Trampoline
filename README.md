# 🤸 TRAMPOLINE — 競技トランポリンゲーム

スマートフォン向け競技トランポリンゲーム。タップとスワイプで技を入力し、難度点と高さを競おう。

## 遊び方

### 予備ジャンプ
- ベッドが緑に光ったタイミング（底）で **タップ** → 高さアップ
- 3回タップで技入力モードへ

### 技コマンド入力
空中にいる間にコマンドを入力します。

| 入力 | 意味 |
|------|------|
| **円スワイプ ○** | 1番目: 抱え込み姿勢 / 2番目以降: 捻り×1 |
| **V字スワイプ <** | 屈伸姿勢 |
| **斜めスワイプ /** | 1番目: 伸身姿勢 / 2番目以降: 捻り×0.5 |
| **タップN回** | 宙返りN回 (最大3回) |

### 例
`○ → タップ×2 → ○ → ○/` = **前方 抱え込み 3回宙返り 2.5捻り**

### ルール
- 前方系→後方系×5セット = 計10本
- 前方系は捻りが必ず0.5の奇数倍（0.5 / 1.5 / 2.5 / 3.5）
- 後方系は捻りが必ず整数（1 / 2 / 3）
- 最大3回宙返り
- 最後にストレートジャンプ → 着地タップでゲーム終了

### 採点
- **難度点 (D-score)**: 技の難しさ
- **高さボーナス**: 演技中の平均高さ
- **着地タイミング**: 着地の精度

## デプロイ (Netlify)

3. ビルド設定不要（静的HTML）→ Deploy
